1. 로컬디스크 C에 폴더`nodejs_pharmacy` 만듭시다
2. C:\nodejs_pharmacy> 경로로 맞추어
```console
npm init
```
끝나면 그 폴더로 파일들을 옮기고
```console
npm install express
```

마지막으로
```console
npm install axios
```
